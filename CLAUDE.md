# Claude Instructions for Paydirt

## TypeScript Style

- No semicolons in any TypeScript file — Prettier is configured with `semi: false`

## Git Workflow

- Commit after every individual change — do not batch multiple changes into one commit
- Keep commit messages to a single sentence
- Push to origin after each commit

## Package Philosophy

- Always prefer the latest available version of packages
- If upgrading requires restructuring, do it — don't stay on old versions for convenience
